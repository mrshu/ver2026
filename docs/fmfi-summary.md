# FMFI UK summary view

The web viewer opens with a focused summary for the three VER 2026 evaluation
areas where FMFI UK appears:

- Matematické vedy
- Fyzikálne vedy
- Informačné a komunikačné vedy

The summary is computed in the browser from `web/data.json`; the values are not
hard-coded from the email screenshot. A row is treated as FMFI UK when the
institution is `Univerzita Komenského v Bratislave` and the institution level
contains `Fakulta matematiky, fyziky a informatiky`.

## Reproduced fields

`Výstupy 1+2`, `Dosah 1+2`, `Prostredie 1+2`, and `Celkový profil 1+2` use the
`__top2` fields generated by `ver2026-dump`. For outputs, this means the sum of
world-class and significant international quality. For impact, environment, and
overall profile, it means the sum of excellent and very good quality.

`Poradie v oblasti` is computed by sorting every institution in the same
evaluation area by `celkovy__score` in ascending order. Lower score is better.

## Local use

Regenerate the JSON after changing the XLSX:

```bash
uv run ver2026-dump
```

Serve the static site from the `web` directory:

```bash
cd web
python3 -m http.server 8765
```

Open <http://localhost:8765/>. The browser needs an HTTP server because the page
fetches `data.json`.
